Understanding Hyperlinks in HTML

In HTML, hyperlinks are created using the <a> (anchor) tag. The href attribute specifies the destination URL. The target attribute specifies where to open the linked document.

This link opens OpenAI in the same tab:

Example 2: Hyperlink with target="_blank"

This link opens OpenAI in a new tab:
Visit OpenAI in New Tab

Different Target Attribute Values

  • _self – Opens the link in the same tab (default).
  • _blank – Opens the link in a new tab or window.
  • _parent – Opens the link in the parent frame.
  • _top – Opens the link in the full body of the window.
